# Описание синтаксиса

## Поле ввода

Заголовок: Значение__.vox {ИмяСвойства:ЗначениеСвойства}

Модификаторы кнопок:

Модификатор|Описание
---|---
v| Кнопка выбора
o| Кнопка открытия
x| Кнопка очистки
.| Кнопка выбора из списка
^| Кнопка регулирования

Примеры:

*Фамилия: Иванов*

*Контрагент: ООО Типы__vo*

## Поле флажка

[X] Флажок</br>
или</br>
Флажок []


Примеры:

*[X] Показать все*

*Показать все [ ]*
